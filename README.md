# Join Frontend

**Join Frontend** is the user interface for the task management app **Join**.  
It provides a Kanban-style board, user management, and task features to help organizations manage work efficiently.  

---

## Functionality

- **Kanban Board**: Visualize tasks in different status columns  
- **User Management**: Register and manage user accounts  
- **Task Assignment**: Assign tasks to specific users  
- **Due Dates**: Set deadlines for tasks  

---

## Installation & Setup

### Requirements
- A functioning Django backend, preferably my [**join-backend**](https://github.com/johannesbraun54/join-backend), which is **not** included in this project.
- Visual Studio Code with the **Live Server** extension or a similar option for launching `index.html` locally in the browser.

### Clone repository
```bash
git clone https://github.com/johannesbraun54/join_frontend.git
cd join_frontend
```

### Usage
1. Ensure that your join-backend is running.
2. Open this project in **Visual Studio Code**.
3. Right-click on the `index.html` file and select **Open with Live Server** to start the project.
